# WebSocket Chat Server

Web-based real-time chat application built with Flask and WebSocket.

## Features
- User authentication
- Password hashing
- Chat rooms
- Invitations
- SQLite database
- Message logging
- Real-time messaging

## Technologies
- Python
- Flask
- Flask-SocketIO
- SQLAlchemy
- SQLite

## Installation

```bash
pip install -r requirements.txt
python app.py

## Usage

Open in browser

http://localhost:5000
